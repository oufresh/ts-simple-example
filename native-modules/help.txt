1. Installare Node.Js
2. Installare typescript con "npm install -g typescript"
3. Inizializzare il progetto nella dircetory voluta con "tsc --init"
4. modificare il file tsconfig.json secondo quanto volute ("sourceMap": true,  "removeComments": true, "noImplicitAny": true,  ecc...)
5. Compilare il typescript con "tsc -sourceMap"
6. Includere nella pagina html il file js generato
